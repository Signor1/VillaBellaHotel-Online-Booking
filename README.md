# Villa Bella Hotels
## Hotel website

An online booking hotel website for booking or making a room reservation. After booking, the site admin can approve or cancel ones booking. Also, bookings are printable as reciept or prove of booking for hotel checking-in-and-out.
## Tech Stack

 *Client** HTML, BootstrapCSS, JavaScript
 *Server** PHP, MySQL
